Descripción
En esta práctica se busca diseñar e implementar una aplicación gráfica para interactuar con
el CRB 15000 desde el Flexpendant. Se diseñará una interfaz gráfica sencilla para acceder
a las entradas/salidas del controlador y poder elegir una acción (e.g. color de la pieza a
recoger).
Objetivos:
❖ Uso de Visual Studio Code (a realizar previo a la sesión en laboratorio)
➢ Descargar el Omnicore App SDK
(https://developercenter.robotstudio.com/omnicore-sdk)
➢ Poner en marcha WebApp de ejemplo
https://github.com/edwinDOS/GOFA-webapp.git
➢ Importar archivo de configuración EIO.cfg del controlador real
➢ Adaptar la WebApp
❖ Prueba en robot real (a realizar en laboratorio)
➢ Descargar WebApp al controlador real
➢ Reiniciar el controlador y FlexPendant
➢ Verificación del funcionamiento
Indicaciones del proceso:
● La interfaz gráfica debe mostrar botones/íconos que permitan seleccionar piezas
individualmente.
● Debe permitir confirmar el inicio de la recogida de la pieza.
● Permitirá indicar el por el usuario el número de piezas a recoger (opcional)
● Indicará visualmente cuando el robot esté ocupado ejecutando la tarea y cuando la
haya finalizado (opcional)
● Deberá indicar el número de piezas que se han recogido (opcional)
Nota: Las mejoras visuales (apariencia/íconos/funciones extra) serán valoradas
positivamente
